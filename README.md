# Shadow RAT

*A high performance rat server and client written in 100% safe rust*

# Build (WSL)

Install rust
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Clone repository
```bash
git clone https://github.com/vu1nerab13/musubi.git
```

Build
```bash
cd musubi
cargo build
```

Run
```bash
cargo run
```
